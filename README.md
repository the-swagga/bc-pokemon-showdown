# Getting Started / Data Collection

To set up and run the project, do the following:
1. Clone this repository.
2. Create a .env file locally and enter your username, password, and format as ```USER=yourusername```, ```PASSWORD=yourpassword```, and ```FORMAT=format```.
3. Run ```poke_env_main.py``` and follow the instructions in the console.

Once the project is running, use the console to queue for battles. Data will be saved to a CSV file after the end of each battle or when a player forfeits.
